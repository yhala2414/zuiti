# Invocation Template

Use `project-collaboration-operating-system` for this project.

Mode: audit / improve / retrospective

Goal:
[describe what you want]

Constraints:

- Do not modify files unless explicitly asked.
- Preserve existing project conventions.
- Classify failures before proposing document or workflow changes.

Output:

- Current collaboration picture
- Key risks or gaps
- Minimal improvement plan
- Verification or follow-up checks

For daily development, put the concrete task in `Goal`. For a retrospective, describe what happened and where the collaboration drifted from expectations.
